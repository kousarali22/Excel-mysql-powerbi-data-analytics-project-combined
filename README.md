A complete end-to-end data analysis project focused on analyzing road accident data to identify key patterns, trends, and factors contributing to casualties. The project involves data cleaning, SQL querying, and interactive dashboard creation using Excel, Power BI, and Tableau.

📌 Project Objective

To analyze road accident data and provide meaningful insights about:

Total Accidents & Casualties

Severity Levels (Fatal, Serious, Slight)

Vehicle Type Analysis

Monthly & Yearly Trends

Location-based Analysis

Road Type & Light Conditions Impact

🛠️ Tools & Technologies Used

📊 Microsoft Excel – Data cleaning, pivot tables, KPI calculations

🗄️ MySQL – Data querying and aggregation using SQL

📈 Power BI – Interactive dashboard & data visualization

📉 Tableau – Trend analysis & dynamic filtering

📂 Project Workflow

Data Cleaning

Removed duplicates and null values

Standardized column names

Created calculated columns (Month, Year, Severity Category)

SQL Analysis

Total casualties by year

Casualties by vehicle type

Severity distribution

Monthly accident trend

Location-wise accident count

Dashboard Creation

KPI Cards (Total Casualties, Total Accidents, Fatal Cases)

Monthly Trend Line Chart

Vehicle Type Breakdown

Road Type Analysis

Slicers for Year & Location

📊 Key Insights

Majority of casualties fall under Slight severity.

Two-wheeler and car accidents contribute significantly to total cases.

Peak accident rates observed in specific months.

Urban areas show higher accident frequency compared to rural regions.

🎯 Business Impact

This dashboard helps:

Identify high-risk accident periods

Understand severity distribution

Support data-driven decision-making for road safety improvements
